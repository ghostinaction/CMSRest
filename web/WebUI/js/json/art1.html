《心理罪之城市之光》曝“蛛丝马迹”版角色海报 审判世道人心 Thu Feb 23 03:45:59 CST 2017

由邓超、阮经天、刘诗诗领衔主演，林嘉欣特别出演，郭京飞友情出演的华语犯罪悬疑巨制《心理罪之城市之光》继超概念海报之后再度发力，曝“蛛丝马迹”版角色海报。该组海报中，神秘人藏身暗处，透过密集的蛛丝窥视前方，而五位演员拨开层层蛛丝马迹，目光所及透露血腥杀机，搭配五个暗藏深意的问题，为片中人物的命运笼罩上一层迷雾。



罪恶城市危机四伏 蛛丝马迹初现端倪


《心理罪之城市之光》本次曝光的角色海报十分特殊，除五位演员的单独款之外，还有一款独立海报，隐藏在黑暗深处的神秘人对整座城市发出了“人在做，谁在看？”的灵魂拷问。而五张角色海报中，五位主人公纷纷以问作答，意有所指穿透蛛丝。在层层谜团之下，他们的眼睛究竟看到了什么？



影片中邓超饰演的方木不再是初出茅庐的大学生，而是经历数次生死劫难后成熟坚定的警察，面对城市中不断发生的罪案，他追逐着真凶也探究其原因，发现种种巧合中早有命定般的联系；阮经天饰演的江亚是命运与悲剧催生的产物，他相信自己才是因果报应的执行者；刘诗诗扮演的女警米楠至诚至真，不断抚慰受伤者的心灵；林嘉欣所扮演的魏巍看起来毫不起眼，弱小的身躯却是疑案峰回路转的关键；而郭京飞饰演的律师任川，虽然偶有妥协却坚信真相终会水落石出，仍难逃暗黑人心的“审判”。五组不一样的回答，也同时提出了五个更难解的问题。而最终的答案，则要等到影片上映后留给观众亲自揭晓。




人物形象颠覆升级 因戏成痴无法自拔




作为《心理罪》系列最新一部，《心理罪之城市之光》中人物背景与性格之复杂，都比前作有了极大提升。对于邓超、阮经天、刘诗诗、林嘉欣、郭京飞而言，都有着不小的挑战。尤其是涉及犯罪悬疑题材，他们不仅要熟知犯罪心理学、法学、法医学、痕迹学等专业知识，对于所出演的角色更是要下十二分精力去钻研。五位演员在杀青后都曾不约而同的表示自己入戏太深，全身心的投入为原本立体饱满的剧情人物赋予了更多的可能性。


作为《心理罪》系列小说的主角，“方木”广受读者的喜爱，此次邓超全面升级出演历经沧桑的壮年时期方木，剧照发布后网友惊呼“认不出”，一头银发与黑框眼镜略显颓废的搭配将这位“犯罪心理推理天才”的气质展露无遗。阮经天出演反派江亚，一名充满争议的变态罪犯，金马影帝大胆突破，他曾表示这是近年来最大的一次挑战，为了塑造江亚的身形枯槁不惜让自己体重骤降。刘诗诗一改往日温婉气质，以颠覆性的干练短发亮相影片，饰演女警察米楠，不仅擅长追踪蛛丝马迹剖析案情，更多次在关键时刻对方木伸出援手。剧中最为神秘的角色魏巍则由金马影后林嘉欣出演，“睡美人”的设定令案情扑朔迷离。实力派演员郭京飞饰演行走在道德与法律平衡木上的律师任川，始终追寻答案的他在剧中挣扎徘徊，极具张力。


电影《心理罪之城市之光》由金牌推手江志强、上影集团董事长任仲伦联手担任监制，鬼才导演徐纪周执导并亲自编剧，《心理罪》系列小说原作者雷米友情客串。
